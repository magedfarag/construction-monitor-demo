# Archive

This folder stores documentation that is worth preserving but should not be confused with the maintained operating docs.

## What Lives Here

| Area | Purpose |
|---|---|
| `notes/` | Tactical one-off investigations, map-fix notes, demo scripts, and testing side-notes |
| `legacy/` | Preserved placeholders or superseded docs that are not part of the active delivery surface |
| `v2/` | Older V2 pilot material retained for reference |

If you are trying to understand or run the current system, start from [../../README.md](../../README.md) and the maintained docs linked from [../README.md](../README.md).
